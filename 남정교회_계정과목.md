## 코드 체계
- **6자리 숫자** (ACCODE)
- **CD**: D=수입(Debit), C=지출(Credit)
- 상위 2자리: 대분류 (위원회/회계단위)
- 상위 3자리 + 000: 중분류 (부서)
- 하위 3자리: 세부 계정

## 코드 구조 요약

```
1xxxxx = 수입 (D)
  11xxxx = 일반회계 수입
  12xxxx = 구제회계 수입
  13xxxx = 건축회계 수입
  14xxxx = 선교회계 수입
  15xxxx = 기타회계 수입

2xxxxx = 지출 (C)
  21xxxx = 서무(행정)위원회
  22xxxx = 교육위원회
  23xxxx = 예배위원회
  24xxxx = 재정위원회
  25xxxx = 관리위원회
  26xxxx = 전도선교위원회
  27xxxx = 봉사,섬김위원회

3xxxxx = 부채
4xxxxx = 기타
```

## 계층 구조 (3단계)
- **대분류** (xx0000): 위원회/회계단위 (예: 210000 서무위원회)
- **중분류** (xxx000): 부서 (예: 211000 서무부)
- **소분류** (xxxxxx): 세부 계정 (예: 211001 교역자사례)
